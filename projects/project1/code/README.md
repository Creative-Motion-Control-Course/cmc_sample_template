# Project 1 — Code

This folder contains the Arduino/C++ source code for Project 1.

## Files

- `project1.ino` — Main Arduino sketch

## Dependencies

- [StepDance Library](https://stepdance.org)

## Uploading

1. Open `project1.ino` in the Arduino IDE.
2. Install the StepDance library if you haven't already.
3. Select your board and port.
4. Click **Upload**.
